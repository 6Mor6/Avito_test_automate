# E2E-тесты таск-трекера (Selenium + pytest + Allure)

## Назначение
Автоматизированные end-to-end тесты для проверки двух ключевых сценариев в таск-трекере:
1. **Создание задачи** — проверка, что задача корректно создаётся и отображается на доске после перезагрузки.
2. **Открытие карточки задачи** — проверка, что при клике на задачу форма редактирования открывается с точным совпадением всех полей (название, описание, проект, приоритет, исполнитель).

## Требования
- Python 3.8+


## Установка зависимостей
```bash
pip install selenium webdriver-manager pytest allure-pytest
```

## Запуск тестов
```bash
pytest tests/ --alluredir=./allure-results
```

## Генерация отчёта Allure

allure serve ./allure-results 

**Отчёт открывается в браузере. Включает** :

* Пошаговые действия
* Скриншоты
* Логи
* Статусы (Passed/Failed)
* Время выполнения
